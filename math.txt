高中数学知识点大全
一、集合与常用逻辑用语
1. 集合
1.1 集合的定义：由确定的对象（元素）组成的整体，元素具有确定性、互异性、无序性。
1.2 元素与集合的关系：∈（属于）、∉（不属于），记法：元素a∈集合A，a∉A。
1.3 集合的表示方法：
 - 列举法：把集合中的元素一一列举出来，如{1,2,3}；
 - 描述法：用集合中元素的共同特征表示，如{x|x是小于5的正整数}；
 - 区间法：适用于实数集的子集，如[1,5)、(-∞,3]；
 - 韦恩图（Venn图）：用封闭图形表示集合间的关系。
1.4 常用数集：
 - 自然数集：N（含0）；正自然数集：N₊/N*（不含0）；
 - 整数集：Z；有理数集：Q；实数集：R；无理数集：∁ᵣQ。
1.5 集合间的关系：
 - 子集：若任意a∈A，则a∈B，记A⊆B（A是B的子集）；
 - 真子集：A⊆B且A≠B，记A⫋B；
 - 相等：A⊆B且B⊆A，记A=B；
 - 空集：∅，是任何集合的子集，是任何非空集合的真子集。
1.6 集合的运算：
 - 交集：A∩B={x|x∈A且x∈B}（两集合的公共元素）；
 - 并集：A∪B={x|x∈A或x∈B}（两集合的所有元素，不重复）；
 - 补集：∁ᵤA={x|x∈U且x∉A}（全集U中不属于A的元素），性质：A∪∁ᵤA=U，A∩∁ᵤA=∅。
1.7 集合运算性质：
 - A∩A=A，A∩∅=∅，A∩B=B∩A；
 - A∪A=A，A∪∅=A，A∪B=B∪A；
 - A∩(B∪C)=(A∩B)∪(A∩C)，A∪(B∩C)=(A∪B)∩(A∪C)；
 - ∁ᵤ(A∩B)=∁ᵤA∪∁ᵤB，∁ᵤ(A∪B)=∁ᵤA∩∁ᵤB（德摩根定律）。
2. 常用逻辑用语
2.1 命题：可以判断真假的陈述句，分为真命题、假命题；不含逻辑联结词的命题叫简单命题，含“且、或、非”的叫复合命题。
2.2 逻辑联结词：
 - 且（∧）：p∧q，只有p、q都为真，命题才为真；
 - 或（∨）：p∨q，只要p、q有一个为真，命题就为真；
 - 非（¬）：¬p，与p的真假相反（否定命题）。
2.3 全称量词与存在量词：
 - 全称量词：∀（任意），全称命题：∀x∈M，p(x)，否定：∃x∈M，¬p(x)；
 - 存在量词：∃（存在），特称命题：∃x∈M，p(x)，否定：∀x∈M，¬p(x)。
2.4 充分条件与必要条件：
 - 若p⇒q（p推出q），则p是q的充分条件，q是p的必要条件；
 - 若p⇔q（p推出q且q推出p），则p是q的充要条件；
 - 若p⇒q但q⇏p，则p是q的充分不必要条件；
 - 若q⇒p但p⇏q，则p是q的必要不充分条件；
 - 若p⇏q且q⇏p，则p是q的既不充分也不必要条件。
2.5 命题的否定与否命题的区别：
 - 命题的否定（非命题）：只否定结论，不否定条件，如“若p，则q”的否定是“若p，则¬q”；
 - 否命题：既否定条件，也否定结论，如“若p，则q”的否命题是“若¬p，则¬q”。
二、函数的概念与基本初等函数
1. 函数的概念
1.1 定义：设A、B是非空的实数集，如果对于集合A中的任意一个数x，按照某种确定的对应关系f，在集合B中都有唯一确定的数y和它对应，那么就称f：A→B为从集合A到集合B的一个函数，记为y=f(x)，x∈A。
1.2 三要素：定义域（x的取值范围）、对应关系（f）、值域（y的取值范围，由定义域和对应关系决定）。
1.3 定义域的求法（核心：使函数有意义）：
 - 分式函数：分母≠0；
 - 偶次根式：被开方数≥0；
 - 对数函数：真数>0，底数>0且≠1；
 - 指数函数：底数>0且≠1，定义域为R；
 - 复合函数：先求内层函数的定义域，再结合外层函数的要求确定整体定义域。
1.4 函数的表示方法：解析法、列表法、图像法。
1.5 函数相等：定义域相同且对应关系相同（与解析式形式无关）。
1.6 分段函数：在定义域的不同区间上，有不同的解析式，注意分段点的取值（不重复、不遗漏），分段函数是一个函数，不是多个函数。
2. 函数的性质
2.1 单调性（增减性）：
 - 增函数：若x₁<x₂，则f(x₁)<f(x₂)（x₁、x₂∈定义域），图像从左到右上升；
 - 减函数：若x₁<x₂，则f(x₁)>f(x₂)（x₁、x₂∈定义域），图像从左到右下降；
 - 判断方法：定义法（取值→作差→变形→判断符号）、导数法（后续章节）、图像法、复合函数单调性（同增异减）。
2.2 奇偶性：
 - 前提：定义域关于原点对称；
 - 奇函数：f(-x)=-f(x)，图像关于原点对称，若x=0有定义，则f(0)=0；
 - 偶函数：f(-x)=f(x)，图像关于y轴对称；
 - 常见结论：奇函数+奇函数=奇函数，偶函数+偶函数=偶函数，奇函数×奇函数=偶函数，偶函数×偶函数=偶函数，奇函数×偶函数=奇函数。
2.3 周期性（选修延伸）：
 - 定义：存在非零常数T，使得对任意x∈定义域，都有f(x+T)=f(x)，则T是函数的一个周期；最小的正周期叫最小正周期；
 - 常见周期函数：正弦、余弦函数（周期2π），正切函数（周期π）。
2.4 最值：
 - 最大值：定义域内存在x₀，使得对任意x∈定义域，都有f(x)≤f(x₀)，则f(x₀)是最大值；
 - 最小值：定义域内存在x₀，使得对任意x∈定义域，都有f(x)≥f(x₀)，则f(x₀)是最小值；
 - 求法：单调性法、配方法、导数法、图像法。
3. 基本初等函数
3.1 一次函数
 - 解析式：y=kx+b（k≠0），当b=0时，为正比例函数y=kx（k≠0）；
 - 定义域、值域：均为R；
 - 单调性：k>0时，增函数；k<0时，减函数；
 - 图像：过点(0,b)和(-b/k, 0)的直线，k为斜率，b为纵截距。
3.2 二次函数
 - 解析式：
  ① 一般式：y=ax²+bx+c（a≠0）；
  ② 顶点式：y=a(x-h)²+k（a≠0），顶点坐标(h,k)；
  ③ 零点式（两根式）：y=a(x-x₁)(x-x₂)（a≠0），x₁、x₂是方程ax²+bx+c=0的两根；
 - 定义域：R；
 - 值域：a>0时，[k, +∞)（k为顶点纵坐标）；a<0时，(-∞, k]；
 - 对称轴：x=-b/(2a)（一般式），x=h（顶点式）；
 - 单调性：a>0时，在(-∞, -b/(2a)]上减，在[-b/(2a), +∞)上增；a<0时，相反；
 - 最值：a>0时，x=-b/(2a)时，y_min=(4ac-b²)/(4a)；a<0时，y_max=(4ac-b²)/(4a)；
 - 零点（与x轴交点）：解方程ax²+bx+c=0，判别式Δ=b²-4ac：
  Δ>0：两个不相等的实根；Δ=0：两个相等的实根；Δ<0：无实根。
3.3 反比例函数
 - 解析式：y=k/x（k≠0）；
 - 定义域：{x|x≠0}；值域：{y|y≠0}；
 - 单调性：k>0时，在(-∞,0)和(0,+∞)上分别减；k<0时，在(-∞,0)和(0,+∞)上分别增；
 - 奇偶性：奇函数，图像关于原点对称；
 - 图像：双曲线，渐近线为x=0（y轴）和y=0（x轴）。
3.4 指数函数
 - 解析式：y=aˣ（a>0且a≠1）；
 - 定义域：R；值域：(0, +∞)；
 - 图像：过点(0,1)，a>1时，增函数；0<a<1时，减函数；
 - 性质：
  ① a⁰=1（a≠0）；a¹=a；
  ② aᵐ·aⁿ=aᵐ⁺ⁿ；aᵐ÷aⁿ=aᵐ⁻ⁿ；(aᵐ)ⁿ=aᵐⁿ；(ab)ⁿ=aⁿbⁿ；
  ③ 当a>1时，x>0则aˣ>1，x<0则0<aˣ<1；当0<a<1时，相反。
3.5 对数函数
 - 定义：若aˣ=N（a>0且a≠1），则x=logₐN（对数），其中a为底数，N为真数；
 - 解析式：y=logₐx（a>0且a≠1），与指数函数y=aˣ互为反函数；
 - 定义域：(0, +∞)；值域：R；
 - 图像：过点(1,0)，a>1时，增函数；0<a<1时，减函数；
 - 对数性质：
  ① 对数恒等式：a^logₐN=N；logₐaˣ=x；
  ② 换底公式：logₐN=log_bN / log_ba（b>0且b≠1）；推论：logₐb=1/log_ba；
  ③ 运算法则：
   logₐ(M·N)=logₐM + logₐN；
   logₐ(M/N)=logₐM - logₐN；
   logₐMⁿ=nlogₐM（n∈R）；
   logₐ√ⁿM=(1/n)logₐM；
 - 常用对数：log₁₀x=lgx；自然对数：logₑx=lnx（e≈2.71828）。
3.6 幂函数
 - 解析式：y=xᵏ（k为常数，k∈R）；
 - 定义域：随k的取值变化，如k=1时定义域为R，k=1/2时定义域为[0,+∞)，k=-1时定义域为{x|x≠0}；
 - 图像：过点(1,1)，k>0时，在(0,+∞)上增；k<0时，在(0,+∞)上减；
 - 常见幂函数：y=x（一次）、y=x²（二次）、y=x³（三次）、y=x^(1/2)（根号x）、y=x⁻¹（反比例）。
4. 函数的图像变换
4.1 平移变换（以y=f(x)为基础）：
 - 左右平移：y=f(x+h)，h>0向左移h个单位，h<0向右移|h|个单位；
 - 上下平移：y=f(x)+k，k>0向上移k个单位，k<0向下移|k|个单位。
4.2 对称变换：
 - 关于x轴对称：y=-f(x)；
 - 关于y轴对称：y=f(-x)；
 - 关于原点对称：y=-f(-x)；
 - 关于直线y=x对称：y=f⁻¹(x)（反函数，前提是函数单调，存在反函数）。
4.3 伸缩变换：
 - 横向伸缩：y=f(ωx)（ω>0），ω>1时，横坐标缩短为原来的1/ω；0<ω<1时，横坐标伸长为原来的1/ω；
 - 纵向伸缩：y=Af(x)（A>0），A>1时，纵坐标伸长为原来的A倍；0<A<1时，纵坐标缩短为原来的A倍。
4.4 翻折变换：
 - 上翻下：y=|f(x)|，将x轴下方的图像翻折到x轴上方，上方图像不变；
 - 左翻右：y=f(|x|)，将y轴右侧的图像翻折到y轴左侧，右侧图像不变（偶函数图像特征）。
三、三角函数与解三角形
1. 三角函数的概念
1.1 任意角：
 - 定义：平面内一条射线绕端点旋转形成的角，正角（逆时针旋转）、负角（顺时针旋转）、零角（不旋转）；
 - 终边相同的角：与角α终边相同的角的集合为{β|β=α+2kπ，k∈Z}；
 - 象限角：角的终边在第几象限，就叫第几象限角；终边在坐标轴上的角不属于任何象限。
1.2 弧度制：
 - 定义：弧长等于半径的弧所对的圆心角为1弧度（rad）；
 - 弧度与角度的换算：180°=π rad，1°=π/180 rad，1 rad=(180/π)°≈57.3°；
 - 弧长公式：l=|α|r（α为弧度，r为半径）；
 - 扇形面积公式：S=1/2lr=1/2|α|r²。
1.3 任意角的三角函数：
 - 定义：设角α的终边上任意一点P(x,y)，到原点的距离为r=√(x²+y²)>0，则：
  sinα=y/r（正弦），cosα=x/r（余弦），tanα=y/x（正切，x≠0）；
 - 三角函数值的符号：
  第一象限：sinα>0，cosα>0，tanα>0；
  第二象限：sinα>0，cosα<0，tanα<0；
  第三象限：sinα<0，cosα<0，tanα>0；
  第四象限：sinα<0，cosα>0，tanα<0；
 - 特殊角的三角函数值（重点）：
  α（弧度）：0，π/6，π/4，π/3，π/2，π，3π/2，2π；
  sinα：0，1/2，√2/2，√3/2，1，0，-1，0；
  cosα：1，√3/2，√2/2，1/2，0，-1，0，1；
  tanα：0，√3/3，1，√3，不存在，0，不存在，0；
 - 同角三角函数基本关系：
  ① sin²α + cos²α=1；
  ② tanα=sinα/cosα（α≠π/2 +kπ，k∈Z）；
  ③ 变形：sin²α=1-cos²α，cos²α=1-sin²α，sinα=tanα·cosα。
1.4 诱导公式（核心：奇变偶不变，符号看象限）：
 - 奇偶性：sin(-α)=-sinα，cos(-α)=cosα，tan(-α)=-tanα；
 - 与π的关系：sin(π±α)=∓sinα，cos(π±α)=-cosα，tan(π±α)=±tanα；
 - 与π/2的关系：sin(π/2±α)=cosα，cos(π/2±α)=∓sinα；
 - 与3π/2的关系：sin(3π/2±α)=-cosα，cos(3π/2±α)=±sinα；
 - 与2π的关系：sin(2π±α)=±sinα，cos(2π±α)=cosα，tan(2π±α)=±tanα。
2. 三角函数的图像与性质
2.1 正弦函数y=sinx
 - 定义域：R；值域：[-1,1]；
 - 周期性：最小正周期2π；
 - 奇偶性：奇函数，图像关于原点对称；
 - 单调性：在[-π/2 + 2kπ，π/2 + 2kπ]（k∈Z）上增；在[π/2 + 2kπ，3π/2 + 2kπ]（k∈Z）上减；
 - 最值：x=π/2 + 2kπ时，y_max=1；x=3π/2 + 2kπ时，y_min=-1；
 - 对称中心：(kπ, 0)（k∈Z）；对称轴：x=π/2 + kπ（k∈Z）。
2.2 余弦函数y=cosx
 - 定义域：R；值域：[-1,1]；
 - 周期性：最小正周期2π；
 - 奇偶性：偶函数，图像关于y轴对称；
 - 单调性：在[2kπ，π + 2kπ]（k∈Z）上减；在[π + 2kπ，2π + 2kπ]（k∈Z）上增；
 - 最值：x=2kπ时，y_max=1；x=π + 2kπ时，y_min=-1；
 - 对称中心：(π/2 + kπ, 0)（k∈Z）；对称轴：x=kπ（k∈Z）。
2.3 正切函数y=tanx
 - 定义域：{x|x≠π/2 + kπ，k∈Z}；值域：R；
 - 周期性：最小正周期π；
 - 奇偶性：奇函数，图像关于原点对称；
 - 单调性：在(-π/2 + kπ，π/2 + kπ)（k∈Z）上单调递增；
 - 无最值；对称中心：(kπ/2, 0)（k∈Z）；无对称轴；
 - 渐近线：x=π/2 + kπ（k∈Z）。
2.4 三角函数的图像变换（y=Asin(ωx+φ)+B，A>0，ω>0）
 - 各参数含义：
  A：振幅，决定值域范围（[-A+B, A+B]）；
  ω：角频率，决定周期（T=2π/ω）；
  φ：相位，决定左右平移（平移量为-φ/ω）；
  B：纵向平移量，决定图像上下移动。
 - 变换步骤（从y=sinx到y=Asin(ωx+φ)+B）：
  方法1：y=sinx → 左右平移|φ|个单位（φ>0左移，φ<0右移）→ y=sin(x+φ) → 横向伸缩为原来的1/ω倍 → y=sin(ωx+φ) → 纵向伸缩为原来的A倍 → y=Asin(ωx+φ) → 上下平移|B|个单位 → 最终图像；
  方法2：y=sinx → 横向伸缩为原来的1/ω倍 → y=sin(ωx) → 左右平移|φ/ω|个单位 → y=sin(ωx+φ) → 纵向伸缩为原来的A倍 → y=Asin(ωx+φ) → 上下平移|B|个单位 → 最终图像（注意平移量差异）。
3. 三角恒等变换
3.1 两角和与差的三角函数公式：
 - sin(α±β)=sinαcosβ±cosαsinβ；
 - cos(α±β)=cosαcosβ∓sinαsinβ；
 - tan(α±β)=(tanα±tanβ)/(1∓tanαtanβ)（α、β、α±β≠π/2 +kπ，k∈Z）。
3.2 二倍角公式：
 - sin2α=2sinαcosα；
 - cos2α=cos²α - sin²α=2cos²α -1=1 - 2sin²α；
 - tan2α=2tanα/(1 - tan²α)（α、2α≠π/2 +kπ，k∈Z）。
3.3 降幂公式（由二倍角公式变形）：
 - sin²α=(1 - cos2α)/2；
 - cos²α=(1 + cos2α)/2；
 - sinαcosα=(sin2α)/2。
3.4 半角公式（选修）：
 - sin(α/2)=±√[(1 - cosα)/2]；
 - cos(α/2)=±√[(1 + cosα)/2]；
 - tan(α/2)=±√[(1 - cosα)/(1 + cosα)]=(1 - cosα)/sinα=sinα/(1 + cosα)。
3.5 辅助角公式（核心）：
 - asinx + bcosx=√(a² + b²)sin(x + φ)，其中tanφ=b/a（φ的象限由a、b的符号决定）；
 - 常用形式：sinx ± cosx=√2sin(x ± π/4)，√3sinx + cosx=2sin(x + π/6)，sinx + √3cosx=2sin(x + π/3)。
4. 解三角形
4.1 三角形的基本关系：
 - 内角和：A + B + C=π（180°）；
 - 边角关系：大边对大角，大角对大边（若a>b，则A>B；反之亦然）；
 - 任意两边之和大于第三边，任意两边之差小于第三边。
4.2 正弦定理：
 - 公式：a/sinA = b/sinB = c/sinC = 2R（R为三角形外接圆半径）；
 - 变形：a=2RsinA，b=2RsinB，c=2RsinC；
 - sinA=a/(2R)，sinB=b/(2R)，sinC=c/(2R)；
 - 适用场景：已知两角及一边、已知两边及其中一边的对角（注意多解情况）。
4.3 余弦定理：
 - 公式：a²=b² + c² - 2bccosA；b²=a² + c² - 2accosB；c²=a² + b² - 2abcosC；
 - 变形：cosA=(b² + c² - a²)/(2bc)；cosB=(a² + c² - b²)/(2ac)；cosC=(a² + b² - c²)/(2ab)；
 - 适用场景：已知两边及夹角、已知三边、已知一边及两边的对角（求角）。
4.4 三角形的面积公式：
 - S=1/2ab sinC=1/2bc sinA=1/2ac sinB（核心，最常用）；
 - S=√[p(p - a)(p - b)(p - c)]（海伦公式，p=(a + b + c)/2）；
 - S=1/2(a + b + c)r（r为三角形内切圆半径）；
 - S=abc/(4R)（R为三角形外接圆半径）。
4.5 解三角形的多解问题（重点易错）：
 - 已知a、b、A（两边及其中一边的对角），判断解的个数：
  ① 当A为锐角时：
   a>b或a=b sinA：1个解；
   b sinA <a <b：2个解；
   a<b sinA：无解；
  ② 当A为直角或钝角时：
   a>b：1个解；
   a≤b：无解。
四、数列
1. 数列的概念
1.1 定义：按一定顺序排列的一列数，叫数列，记为{aₙ}，其中aₙ是第n项（通项）。
1.2 通项公式：如果数列{aₙ}的第n项与n之间的关系可以用一个公式表示，这个公式叫通项公式（不一定唯一）。
1.3 递推公式：给出数列的首项（或前几项），以及相邻项之间的关系，叫递推公式（如aₙ₊₁=2aₙ +1）。
1.4 数列的前n项和：Sₙ=a₁ + a₂ + ... + aₙ，通项与前n项和的关系：
 - a₁=S₁；
 - aₙ=Sₙ - Sₙ₋₁（n≥2）；
  注意：验证n=1时，若a₁=S₁，则通项公式统一；若不相等，需分段表示。
1.5 数列的分类：
 - 按项数：有穷数列、无穷数列；
 - 按单调性：递增数列（aₙ₊₁>aₙ）、递减数列（aₙ₊₁<aₙ）、常数列（aₙ₊₁=aₙ）、摆动数列。
2. 等差数列
2.1 定义：从第2项起，每一项与它的前一项的差等于同一个常数，这个常数叫公差，记为d（aₙ₊₁ - aₙ=d，n∈N*）。
2.2 通项公式：aₙ=a₁ + (n - 1)d（a₁为首项，d为公差）；
  变形：aₙ=aₘ + (n - m)d（n、m∈N*）。
2.3 前n项和公式：
 - Sₙ=n(a₁ + aₙ)/2（已知a₁、aₙ、n）；
 - Sₙ=na₁ + n(n - 1)d/2（已知a₁、d、n）。
2.4 等差数列的性质：
 - 若m + n = p + q（m、n、p、q∈N*），则aₘ + aₙ = aₚ + a_q；
 - 若m + n = 2k（m、n、k∈N*），则aₘ + aₙ = 2a_k（a_k为中间项）；
 - 数列{aₙ}是等差数列 ⇔ 2aₙ₊₁=aₙ + aₙ₊₂（n∈N*）；
 - 等差数列的前n项和Sₙ是关于n的二次函数（d≠0），且常数项为0；
 - 公差d的求法：d=aₙ - aₙ₋₁（n≥2），d=(aₙ - a₁)/(n - 1)，d=(aₘ - aₙ)/(m - n)。
2.5 等差中项：若a、A、b成等差数列，则A=(a + b)/2，A叫a、b的等差中项。
3. 等比数列
3.1 定义：从第2项起，每一项与它的前一项的比等于同一个常数，这个常数叫公比，记为q（q≠0，aₙ₊₁/aₙ=q，n∈N*）。
3.2 通项公式：aₙ=a₁qⁿ⁻¹（a₁为首项，q为公比）；
  变形：aₙ=aₘqⁿ⁻ᵐ（n、m∈N*）。
3.3 前n项和公式：
 - 当q=1时，Sₙ=na₁（常数列）；
 - 当q≠1时，Sₙ=a₁(1 - qⁿ)/(1 - q)=a₁(qⁿ - 1)/(q - 1)（已知a₁、q、n）；
  变形：Sₙ=(a₁ - aₙq)/(1 - q)（已知a₁、aₙ、q）。
3.4 等比数列的性质：
 - 若m + n = p + q（m、n、p、q∈N*），则aₘ·aₙ = aₚ·a_q；
 - 若m + n = 2k（m、n、k∈N*），则aₘ·aₙ = a_k²（a_k为中间项）；
 - 数列{aₙ}是等比数列 ⇔ aₙ₊₁²=aₙ·aₙ₊₂（n∈N*，aₙ≠0）；
 - 等比数列的各项均不为0，公比q≠0；
 - 公比q的求法：q=aₙ/aₙ₋₁（n≥2，aₙ₋₁≠0），q=(aₙ/a₁)^(1/(n-1))，q=(aₙ/aₘ)^(1/(n-m))。
3.5 等比中项：若a、G、b成等比数列，则G²=ab（G≠0），G叫a、b的等比中项。
4. 数列的求和方法（重点）
4.1 公式法：直接用等差数列、等比数列的前n项和公式；
4.2 分组求和法：将数列拆分为两个或多个等差数列、等比数列，分别求和再相加（如aₙ=2ₙ + 3ⁿ）；
4.3 裂项相消法：将通项拆分为两项之差，求和时相互抵消（核心方法），常见裂项形式：
 - 1/[n(n + 1)]=1/n - 1/(n + 1)；
 - 1/[n(n + k)]=(1/k)(1/n - 1/(n + k))（k为常数）；
 - 1/(√n + √(n + 1))=√(n + 1) - √n；
 - 1/[(2n - 1)(2n + 1)]=(1/2)(1/(2n - 1) - 1/(2n + 1))。
4.4 错位相减法：适用于“等差数列×等比数列”型数列（如aₙ=n·2ⁿ），步骤：
① 写出Sₙ=a₁ + a₂ + ... + aₙ；
 ② 两边同乘等比数列的公比q；
  ③ 两式相减，转化为等比数列求和；
  ④ 整理得Sₙ。
4.5 倒序相加法：适用于首尾对称的数列（如等差数列前n项和公式的推导）。
五、立体几何
1. 空间几何体的结构
1.1 多面体
 - 棱柱：有两个面互相平行（底面），其余各面都是四边形，且每相邻两个四边形的公共边互相平行，分为直棱柱（侧棱垂直底面）、斜棱柱（侧棱不垂直底面）、正棱柱（底面为正多边形的直棱柱）；
  性质：侧棱平行且相等，底面全等，侧面为平行四边形（直棱柱侧面为矩形）；
 体积：V=S底·h（h为高）。
 - 棱锥：有一个面是多边形（底面），其余各面都是有一个公共顶点的三角形，分为正棱锥（底面为正多边形，顶点在底面的射影为底面中心）；
  性质：正棱锥的侧棱相等，侧面为全等的等腰三角形；
  体积：V=(1/3)S底·h（h为高）。
 - 棱台：用平行于棱锥底面的平面去截棱锥，底面与截面之间的部分，分为正棱台（由正棱锥截得）；
  性质：正棱台的侧棱相等，侧面为全等的等腰梯形；
  体积：V=(1/3)h(S上 + S下 + √(S上·S下))（h为高，S上、S下为上下底面面积）。
1.2 旋转体
 - 圆柱：以矩形的一边为轴旋转一周得到的几何体，轴为母线的垂线；
  性质：底面为两个全等的圆，侧面展开图为矩形（长为底面圆周长，宽为母线长l）；
  表面积：S=2πr² + 2πrl（r为底面半径，l为母线长）；
  体积：V=πr²h（h为高，直圆柱h=l）。
 - 圆锥：以直角三角形的一条直角边为轴旋转一周得到的几何体
